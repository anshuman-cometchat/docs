## What Changed Across All Files

Three types of additions were made consistently across most/all files:

1. **AI Agent Quick Reference blocks** — `{/* TL;DR for Agents and Quick Reference */}` + `<Info>` blocks with copy-paste-ready code snippets at the top of each page
2. **Response/Error accordions** — `<Accordion title="Response">` and `<Accordion title="Error">` after every SDK method call, documenting success objects and error parameters with sample values
3. **Next Steps navigation** — `<CardGroup>` with `<Card>` links to related pages at the bottom
4. **Description metadata** — `description:` added to frontmatter for SEO
5. **Warnings** — `<Warning>` blocks for common pitfalls (e.g., calling methods before init)
