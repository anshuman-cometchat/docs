The **CometChat UI Kit Builder** for Next.js is a powerful solution designed to seamlessly integrate chat functionality into applications. It provides a robust set of **prebuilt UI components** that are **modular, customizable, and highly scalable**, allowing developers to accelerate their development process with minimal effort.

<Info>
  **Quick Reference for AI Agents & Developers** - **What this page covers**:
  Overview of the CometChat UI Kit Builder for Next.js — features, live demo,
  and how to get started. - **Prerequisites**: A CometChat account and a Next.js
  project. - **Key action**: Explore the features below, then follow the
  [Integration Guide](/chat-builder/nextjs/integration) to set up the Builder in
  your app.
</Info>

---

## **Why Choose CometChat UI Kit Builder?**

- **Rapid Integration** – Prebuilt UI components for faster deployment.
- **Customizable & Flexible** – Modify the UI to align with your brand's identity.
- **Cross-Platform Compatibility** – Works seamlessly across various React-based frameworks.
- **Scalable & Reliable** – Built on CometChat's **robust chat infrastructure** for enterprise-grade performance.

## **Supported Frameworks**

The UI Kit Builder works seamlessly with popular React-based frameworks:

- **React** (Vite, Create React App)
- **Next.js**
- **React Router**

---

## **How It Works**

1. **Design** – Customize your chat UI in the visual builder
2. **Export** – Download production-ready React code
3. **Integrate** – Drop into your app and go live
