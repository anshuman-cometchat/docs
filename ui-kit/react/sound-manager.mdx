Control UI Kit sound cues for calls and messages. This page is for developers who want to play, override, or stop UI Kit sounds programmatically.

## When to use this

- You want to play default UI Kit sounds for calls or messages.
- You want to replace default sounds with custom MP3 files.
- You want to pause or stop ongoing sound playback.
- You need to align sound behavior with your app’s UX.

## Prerequisites

- CometChat React UI Kit v6 installed: `@cometchat/chat-uikit-react`.
- User interaction may be required before audio can play due to browser autoplay policies.

## Quick start

1. Import the sound manager in any module.

```ts
import { CometChatSoundManager, Sound } from "@cometchat/chat-uikit-react";
```

What this does: Makes the sound manager and sound constants available.

2. Play a default sound.

```ts
import { CometChatSoundManager, Sound } from "@cometchat/chat-uikit-react";

CometChatSoundManager.play(Sound.incomingCall);
```

What this does: Plays the default incoming call sound.

3. Play a custom sound file.

```ts
import { CometChatSoundManager, Sound } from "@cometchat/chat-uikit-react";

CometChatSoundManager.play(Sound.incomingCall, "path/to/custom.mp3");
```

What this does: Plays a custom MP3 for the incoming call event.

4. Pause the current sound.

```ts
import { CometChatSoundManager } from "@cometchat/chat-uikit-react";

CometChatSoundManager.pause();
```

What this does: Stops the currently playing sound.

## Core concepts

- Sound events: `Sound.incomingCall`, `Sound.outgoingCall`, `Sound.incomingMessage`, `Sound.outgoingMessage`.
- CometChatSoundManager: Helper class for playing or pausing UI Kit sounds.
- Custom sound path: Provide an MP3 file path to override the default sound.

## Implementation

### Play a default sound

What you’re changing: Trigger a built-in UI Kit sound.
File: Any module (for example `src/sound.ts`)
Applies to: `CometChatSoundManager.play(sound)`
Default behavior: Plays the built-in sound for the event.
Override: Use a custom file path to replace the sound.
Verify: The sound plays after a user interaction event.

```ts
import { CometChatSoundManager, Sound } from "@cometchat/chat-uikit-react";

export function playIncomingCallSound() {
  CometChatSoundManager.play(Sound.incomingCall);
}
```

What this does: Plays the default incoming call sound.

### Play a custom sound

What you’re changing: Replace a default sound with a custom MP3.
File: Any module (for example `src/sound.ts`)
Applies to: `CometChatSoundManager.play(sound, filePath)`
Default behavior: Uses default UI Kit sounds.
Override: Provide a custom MP3 file path.
Verify: Your custom file plays instead of the default.

```ts
import { CometChatSoundManager, Sound } from "@cometchat/chat-uikit-react";

export function playCustomIncomingCallSound() {
  CometChatSoundManager.play(Sound.incomingCall, "/audio/custom-incoming.mp3");
}
```

What this does: Plays a custom MP3 for incoming call events.

### Pause the current sound

What you’re changing: Stop a currently playing sound.
File: Any module (for example `src/sound.ts`)
Applies to: `CometChatSoundManager.pause()`
Default behavior: Sounds continue until finished.
Override: Call pause when you need to stop playback.
Verify: The sound stops immediately.

```ts
import { CometChatSoundManager } from "@cometchat/chat-uikit-react";

export function stopSound() {
  CometChatSoundManager.pause();
}
```

What this does: Stops any sound currently playing.

## Common pitfalls and fixes

| Symptom | Cause | Fix |
| --- | --- | --- |
| No sound plays | Browser autoplay policy blocks audio | Require a user interaction before playing sound. |
| Custom sound not playing | Invalid file path or unsupported format | Use a valid MP3 path and ensure the file loads. |
| Sound keeps playing | `pause()` not called | Call `CometChatSoundManager.pause()` to stop playback. |

## FAQ

**Do I need to initialize anything before playing sounds?**
No. Import `CometChatSoundManager` and call `play()`.

**Which formats are supported for custom sounds?**
Use MP3 files for best compatibility.
