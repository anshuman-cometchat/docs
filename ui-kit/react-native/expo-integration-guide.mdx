## Overview

The CometChat React Native UI Kit does not work with Expo Go since it requires custom native modules. Expo also does not recommend using Expo Go for building production-grade apps.

To use the UI Kit in an Expo app, you need to use [development builds](https://docs.expo.dev/develop/development-builds/introduction/). Follow the official [Expo local app development guide](https://docs.expo.dev/guides/local-app-development/) for detailed instructions.

<Warning>
Expo Go is not supported. You must use development builds to run the CometChat UI Kit.
</Warning>

---

## Configuration

The UI Kit requires camera and microphone permissions for calling features. Add the following to your `app.json` file:

<Tabs>
<Tab title="app.json">
```json
{
  "expo": {
    "ios": {
      "infoPlist": {
        "NSCameraUsageDescription": "This is for Camera permission",
        "NSMicrophoneUsageDescription": "This is for Microphone permission"
      }
    },
    "android": {
      "permissions": [
        "android.permission.INTERNET",
        "android.permission.CAMERA",
        "android.permission.MODIFY_AUDIO_SETTINGS",
        "android.permission.RECORD_AUDIO",
        "android.permission.ACCESS_NETWORK_STATE"
      ]
    }
  }
}
```
</Tab>
</Tabs>
