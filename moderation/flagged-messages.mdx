## Overview

The Flagged Messages endpoint in the Moderation Service API enables app owners and collaborators to review messages that have been flagged for potentially violating moderation rules. Messages can be flagged automatically by the moderation engine based on predefined rules, or manually by moderators who find certain content inappropriate or concerning.

### List Flagged Messages

Retrieves the list of messages that have been flagged by the moderation system or manually by moderators.
<Frame>
  <img src="/images/1600d6b8-moderation_list_flagged_messages-1a2b3c4d5e6f7g8h9i0j.png" />
</Frame>

You can also set this up from your end using the [List Flagged Messages REST API](/moderation/).

---

### Approve Flagged Message

Allows the approval of messages that were previously flagged. Approving a flagged message indicates that it has been reviewed and is considered acceptable according to platform guidelines.

<Frame>
  <img src="/images/1600d6b8-moderation_approve_flagged_message-2b3c4d5e6f7g8h9i0j1a.png" />
</Frame>

You can also set this up from your end using the [Approve Flagged Message REST API](/moderation/).

---

### Block Flagged Message

Allows the blocking of messages that were flagged. Blocking a flagged message will automatically mark it as reviewed.

<Frame>
  <img src="/images/1600d6b8-moderation_block_flagged_message-3c4d5e6f7g8h9i0j1a2b.png" />
</Frame>

You can also set this up from your end using the [Block Flagged Message REST API](/moderation/).
