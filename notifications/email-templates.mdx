---
title: "Email Templates"
---

Use these payload fields when rendering your own email bodies. No sounds apply to email.

## Payload shapes

### One-on-one
```json
{
  "to": { "uid": "cometchat-uid-1", "email": "andrew@example.com", "name": "Andrew Joseph" },
  "messages": [
    {
      "sender": { "uid": "cometchat-uid-4", "avatar": "https://assets.cometchat.io/sampleapp/v2/users/cometchat-uid-4.webp", "name": "Susan Marie" },
      "message": "Are we meeting on this weekend?",
      "messageObject": { "category": "message", "type": "text" }
    },
    {
      "sender": { "uid": "cometchat-uid-4", "avatar": "https://assets.cometchat.io/sampleapp/v2/users/cometchat-uid-4.webp", "name": "Susan Marie" },
      "message": "📷 Has shared an image"
    }
  ],
  "senderDetails": { "uid": "cometchat-uid-4", "name": "Susan Marie", "avatar": "https://assets.cometchat.io/sampleapp/v2/users/cometchat-uid-4.webp" },
  "subject": "New messages from Susan Marie"
}
```

### Group
```json
{
  "to": { "uid": "cometchat-uid-1", "name": "Andrew Joseph" },
  "messages": [
    { "sender": { "uid": "cometchat-uid-5", "name": "John Paul" }, "message": "Hello all! What's up?" },
    { "sender": { "uid": "cometchat-uid-4", "name": "Susan Marie" }, "message": "This is the place I was thinking about" }
  ],
  "groupDetails": { "guid": "cometchat-guid-1", "name": "Hiking Group" }
}
```
