---
title: "Overview"
description: "Manage friend relationships between users in your CometChat app."
---

The **Friends** API manages bidirectional friend relationships between users. When User A adds User B as a friend, both users become friends with each other automatically.

### Key behaviors

- Friendships are **bidirectional** — adding a friend creates the relationship for both users and counts toward both users' friend limits.
- A user can have up to **1,000 friends**.
- Friends lists can be used to filter users in chat UIs or show online friends.

### Relationships

- **Users** — Friends are [Users](/rest-api/users). The `onBehalfOf` header or path UID specifies which user's friend list to manage.

### Error handling

| Error Code            | Description                       |
| --------------------- | --------------------------------- |
| `ERR_UID_NOT_FOUND`   | The specified user does not exist |
| `ERR_ALREADY_FRIENDS` | The users are already friends     |

For the complete list of error codes, see [Error Guide](/articles/error-guide).
